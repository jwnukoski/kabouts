Features

Kabouts main goal is to help users find their way around a small location, and quickly find what they're looking for.

Kabouts currently provides these features:

  • SQL backend for storing many a locations info
    • Store many 'items' for a single X/Y position
  • Add and remove items from a list
  • Visual path finding for each item within a location

Some features still being worked on:

  • Multiple levels
  • Visual popup of items when hovering over a block

Kabouts is wrapped in Electron, so it can be ran independently of a browser.
